# Tutor Platform

A full-stack tutoring platform inspired by services like Revolution Prep, designed to help tutors manage students, track tutoring hours, and handle scheduling.

## Tech Stack

- **Next.js (App Router)** – Full-stack React framework
- **TypeScript** – Type-safe development
- **PostgreSQL (Neon)** – Serverless cloud database
- **Prisma ORM** – Database modeling and queries
- **Auth.js** – Secure authentication and session management
- **Vercel** – Deployment and hosting

## Features

- User authentication (signup / login)
- Secure password hashing
- PostgreSQL database with Prisma ORM
- Persistent user accounts
- Server-side API routes with Next.js

## Status

🚧 Actively in development. This project is being built incrementally to explore full-stack architecture, authentication flows, and scalable SaaS patterns.

Future features will include tutoring session scheduling, hour tracking, and a dashboard for managing students and sessions.
